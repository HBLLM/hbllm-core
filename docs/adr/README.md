## ADR Guidelines

- All architectural decision records follow the Markdown Architectural Decision Records (MADR) format.
- ADRs document significant design choices, rationale, alternative solutions considered, and downstream consequences.
- New ADRs must be submitted via PR and reviewed by core maintainers.
